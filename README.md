## 🧑‍💻 Về dự án SmartCV

Trong 4 tuần cuối của kỳ thực tập, nhóm **TechTitans** đã xây dựng **SmartCV** — hệ thống theo dõi và tối ưu hóa hồ sơ ứng tuyển ứng dụng AI, triển khai trên kiến trúc **AWS Serverless** (Cognito, API Gateway, Lambda, DynamoDB, S3, Amplify, Bedrock). Vai trò cá nhân: **AI/ML & Backend** — tích hợp Amazon Bedrock, xây dựng các Lambda function (`insights`, `settings`, `notes`, `digest`, `followup`) và bộ kiểm thử tự động (207 test case, coverage ≥ 90%).
