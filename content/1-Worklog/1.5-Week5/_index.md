### Week 5 Achievements:

* Successfully deployed and validated a **VPC Peering** connection between two VPCs, confirming private IP connectivity and cross-peer DNS resolution.

* Deployed **AWS Transit Gateway** with attachments and route propagation, simplifying routing across several VPCs.

* Completed the **Hybrid DNS with Route 53 Resolver** lab, including Inbound/Outbound Endpoints, Resolver Rules, and an AWS Managed Microsoft AD, and confirmed two-way DNS resolution using nslookup.

* Deployed a full **Amazon VPC and Site-to-Site VPN** environment, covering subnets, Internet Gateway, NAT Gateway, and Security Groups.

* Practiced using **Reachability Analyzer** to diagnose networking issues involving ENIs, Security Groups, and Network ACLs.

* Built several **AWS Budgets** types (Cost, Usage, Reserved Instance, Savings Plan) to strengthen cost-governance skills.
